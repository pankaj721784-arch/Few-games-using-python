# Python Arcade: 4 Classic Games 🎮

Welcome to **Python Arcade**, a collection of four classic, retro-style arcade games built entirely from scratch using Python and the **Tkinter** GUI library. This repository is a perfect showcase of Python fundamentals, object-oriented programming (OOP), and event-driven GUI development.

---

## 🚀 Featured Games

### 1. Breakout Game
The classic brick-breaking challenge. Control the paddle, bounce the ball, and destroy all the bricks on the screen without letting the ball fall.
* **Key Features:** Collision detection, score tracking, and increasing difficulty.

### 2. Flappy Bird Game
Test your reflexes! Guide the bird through gaps in a series of green pipes. One wrong move, and it's game over.
* **Key Features:** Gravity physics simulation, procedural obstacle generation, and high-score saving.

### 3. Snake Game
The timeless arcade staple. Navigate the snake to eat food and grow longer, but make sure you don't crash into the walls or your own tail!
* **Key Features:** Smooth grid-based movement, dynamic length tracking, and responsive keyboard controls.

### 4. Space Shooter Game
Defend the galaxy! Take control of a spaceship, dodge oncoming asteroids or enemy ships, and shoot them down to rack up points.
* **Key Features:** Sprite animations, rapid-fire mechanics, and multi-entity collision management.

---

## 🛠️ Built With

* **Language:** Python 3.x
* **GUI Framework:** Tkinter (Python's standard GUI toolkit)

*No heavy external game engines (like Pygame) were used, demonstrating the power and versatility of Tkinter's Canvas widget for 2D game rendering.*
